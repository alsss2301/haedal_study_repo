# 바닥공사

 
  

### 1*2의 타일과 2*1의 색칠할 수 있는 경우의 수는 같다.세로가 2이기 때문이다. 따라서 위의 두 방법을 수를 2배 하고, 2*2의 정사각형을 칠할 수 있는 경우를 더하면 된다.  



### 소스 코드

```
n=int(input())
dp=[0]*1001

dp[1]=1
dp[2]=3

for i in range(3,n+1):
    dp[i]=(2*dp[i-2]+dp[i-1])%796796

print(dp[n])


```
