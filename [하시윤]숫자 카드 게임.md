# 숫자 카드 게임

### 풀이 과정 
각 행의 최솟값들 중 최댓값을 구하는 문제입니다.
행의 최솟값을 res와 비교하며 res에 최댓값을 넣습니다.

### 소스 코드

```python

N,M=map(int,input().split())
res=0

for i in range(N):
    card=list(map(int, input().split()))
    min_value=min(card)
    if(min_value>res):
        res=min_value
        
print(res)
